📦 Instalação e Execução
Para rodar o projeto localmente, siga os passos abaixo:

Pré-requisitos
Node.js instalado (LTS recomendada).

Gerenciador de pacotes (npm, yarn ou pnpm).

Aplicativo Expo Go instalado no seu smartphone (Android ou iOS).

Passo a Passo
Clone o repositório:

Bash
git clone https://github.com/seu-usuario/dock.git
cd dock
Instale as dependências:

Bash
npm install
# ou
yarn install
Inicie o servidor do Expo:

Bash
npx expo start
Execução:

Abra o Expo Go no seu celular e escaneie o QR Code que aparecerá no terminal.

Ou pressione a para Android ou i para iOS (caso possua emuladores configurados).
